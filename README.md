## Проектная работа Mesto

Сервис Mesto — интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки.
Пользоваться сервисом могут только пользователи имеющие токен авторизации. На странице представлена следующая информация о текущем пользователе: имя, сфера деятельности и аватар. Все указанные данные могут быть отредактированы пользователем.

[Демо-сайт](https://p-vlasenko.github.io/mesto-project-ff/)

### Описание

Проект выполнен на нативном javascript. Верстка и стили описаны по методологии БЭМ (Блок, Элемент, Модификатор).
